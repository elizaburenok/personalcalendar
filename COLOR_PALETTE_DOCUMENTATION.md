## Notes

1. **Token Structure**: All color tokens follow the `Primitive/{Color Name}` naming convention from Figma.

2. **CSS Variable Naming**: CSS variables are converted from Figma token names using lowercase with hyphens:
   - `Primitive/Primary` → `--primitive-primary`
   - `Primitive/Neutral 1 On Inverse` → `--primitive-neutral-1-on-inverse`

3. **Fixed Variants**: "Fixed" variants maintain the same hex value as their base color but are intended to remain constant in different theme contexts.

4. **On Inverse Variants**: "On Inverse" variants are colors designed to be used on dark/inverse backgrounds.

5. **Total Colors**: This palette contains **42 unique color variables** organized into:
   - 4 Primary colors (base + 3 variants)
   - 4 Secondary colors (base + 3 variants)
   - 16 Neutral colors (4 levels × 4 variants each)
   - 9 Semantic colors (Brand, Success, Error, Warning - with variants)
   - 2 Default colors (base + fixed)

6. **Usage Pattern**: Following the typography token structure, colors should be used with CSS variables and fallback values:
   ```css
   color: var(--primitive-primary, #191919);
   ```
